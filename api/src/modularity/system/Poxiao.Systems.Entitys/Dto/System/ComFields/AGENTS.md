# ComFields (Dto)

## Purpose
公共字段（COM Fields）DTO。可视化开发引擎中可复用的"列字段"配置（标签、类型、校验、默认值等），用于跨表复用同一字段定义。

## Key Files
| File | Description |
|------|-------------|
| `ComFieldsCrInput.cs` / `ComFieldsUpInput.cs` | 创建/更新公共字段 |
| `ComFieldsInfoOutput.cs` | 详情输出 |
| `ComFieldsListOutput.cs` | 列表输出 |
