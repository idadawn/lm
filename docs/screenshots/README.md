## 🛠️ 自动化脚本

如需重新生成其他截图，可使用以下脚本：

```bash
# 主截图脚本（基础页面）
node scripts/screenshot-v5.js

# 弹窗截图脚本
node scripts/screenshot-modals.js

# 补充截图脚本
node scripts/screenshot-final.js

# 修复特定截图
node scripts/fix-missing.js
node scripts/fix-05-09.js
```

**注意**：脚本需要在已安装 Playwright 的环境下运行。
