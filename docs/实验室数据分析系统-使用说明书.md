## 一、系统概述

检测室数据分析系统（Laboratory Data Analysis System）是一款面向实验室检测数据管理的综合平台，主要功能包括：

- **基础数据管理**：产品规格、单位、特性等级、外观特性等基础数据维护
- **数据导入与处理**：支持原始数据、中间数据、磁性数据的导入和管理
- **智能计算与判定**：基于公式配置自动计算检测指标，智能判定质量等级
- **数据分析与报表**：生产驾驶舱可视化展示、月度质量报表统计分析
- **AI辅助功能**：外观特性语义匹配、智能推荐等

### 系统界面概览

![系统界面概览](./screenshots/01-系统界面概览.png)

系统采用左侧导航菜单 + 右侧内容区的经典布局：

- **顶部栏**：显示系统名称、用户信息、消息通知
- **左侧菜单**：功能模块导航，支持展开/收起
- **内容区**：各功能模块的具体操作界面
- **页签栏**：支持多页签切换，方便并行操作

### 2.2 生产驾驶舱（首页）

![生产驾驶舱](./screenshots/03-生产驾驶舱.png)

## 三、基础管理

基础管理模块用于维护系统运行所需的基础数据，是整个系统的数据基石。

### 3.1 产品规格定义

**功能说明**：管理产品规格信息，包括产品编码、名称、检测列数配置以及扩展属性。

![产品规格定义-列表页](./screenshots/基础管理/03-01-产品规格定义-列表页.png)

**操作步骤**：

1. 进入【基础管理】→【产品规格定义】

2. **新增产品规格**：
   - 点击【新增】按钮
   
   ![产品规格定义-新增弹窗](./screenshots/基础管理/03-02-产品规格定义-新增弹窗.png)
   
   - 填写产品编码（必填，唯一标识）
   - 填写产品名称（必填）
   - 设置检测列数（决定该产品规格下检测数据的列数，默认15列）
   - 添加扩展属性（可选）：点击【添加属性】，设置属性名、属性值、单位
   - 点击【确定】保存

3. **编辑产品规格**：
   - 在卡片列表中找到目标产品
   - 点击卡片右上角的【编辑】链接
   - 修改相关信息后保存

4. **删除产品规格**：
   - 点击卡片右上角的【删除】链接
   - 确认删除操作（注意：删除后不可恢复）

5. **排序调整**：
   - 拖动卡片左侧的拖拽图标可调整排序
   - 排序变更会自动保存

**注意事项**：
- 产品编码必须唯一，不能重复
- 检测列数影响数据导入时的列数解析
- 版本标签显示当前生效的版本号

### 3.2 单位管理

**功能说明**：管理物理单位维度及单位定义，支持单位换算和标准化。

![单位管理-单位维度页签](./screenshots/基础管理/03-03-单位管理-单位维度.png)

**界面说明**：
- 单位维度：定义物理量的类别，如长度、重量、时间等
- 单位定义：具体的计量单位，如毫米(mm)、千克(kg)等

**操作步骤**：

1. 进入【基础管理】→【单位管理】

2. **新增单位维度**：
   - 点击【新增单位维度】按钮
   
   ![单位管理-新增维度弹窗](./screenshots/基础管理/03-04-单位管理-新增维度弹窗.png)
   
   - 填写维度名称（如"长度"、"重量"）
   - 填写维度编码（英文标识）
   - 填写描述信息（可选）
   - 点击【确定】保存

3. **新增单位**：
   - 切换到"单位定义"标签页
   
   ![单位管理-单位定义页签](./screenshots/基础管理/03-05-单位管理-单位定义.png)
   
   - 点击【新增单位】按钮
   
   ![单位管理-新增单位弹窗](./screenshots/基础管理/03-06-单位管理-新增单位弹窗.png)
   
   - 选择所属维度
   - 填写单位名称（如"毫米"）
   - 填写单位符号（如"mm"）
   - 设置精度（小数位数）
   - 设置与基准单位的换算比例
   - 如需设为基准单位，勾选"基准单位"选项
   - 点击【确定】保存

**注意事项**：
- 每个维度必须有且只有一个基准单位
- 基准单位不能删除
- 换算比例是基于基准单位的倍数关系

### 3.4 外观特性大类管理

**功能说明**：管理外观特性的大类分类，支持多级分类结构。

![外观特性大类管理-列表页](./screenshots/基础管理/03-09-外观特性大类管理-列表页.png)

**操作步骤**：

1. 进入【基础管理】→【外观特性大类管理】

2. **新增大类**：
   - 点击【新增大类】按钮
   
   ![外观特性大类管理-新增弹窗](./screenshots/基础管理/03-10-外观特性大类管理-新增弹窗.png)
   
   - 填写大类名称
   - 选择父级分类（如不选择则为顶级分类）
   - 填写描述信息
   - 点击【确定】保存

3. **编辑大类**：
   - 在表格中找到目标大类
   - 点击【编辑】按钮修改

### 3.5 外观特性管理

**功能说明**：管理具体的外观特性定义，包括特性名称、关键词、所属大类和等级。

![外观特性管理-主界面](./screenshots/基础管理/03-11-外观特性管理-主界面.png)

**界面布局**：
- 左侧：特性大类树形结构
- 中间：特性定义表格
- 右侧：语义匹配测试面板（可收起）

**操作步骤**：

1. 进入【基础管理】→【外观特性管理】

2. **新增特性**：
   - 点击【新增特性】按钮
   
   ![外观特性管理-新增弹窗](./screenshots/基础管理/03-12-外观特性管理-新增弹窗.png)
   
   - 填写特性名称
   - 选择所属大类
   - 选择特性等级
   - 添加匹配关键词（用于AI匹配）
   - 点击【确定】保存

3. **编辑特性**：
   - 在表格中找到目标特性
   - 点击【编辑】按钮修改

4. **删除特性**：
   - 点击【删除】按钮确认删除

5. **语义匹配测试**：
   - 点击右侧【语义匹配测试】按钮展开测试面板
   
   ![外观特性管理-语义匹配测试](./screenshots/基础管理/03-13-外观特性管理-语义匹配测试.png)
   
   - 在输入框中输入测试文本（如炉号后缀或备注信息）
   - 点击【规则 + AI 匹配】按钮
   - 查看匹配结果和匹配方式

**匹配方式说明**：
- **特性名称精确匹配**：输入文本与特性名称完全一致
- **关键词匹配**：输入文本包含特性的关键词
- **AI分析**：通过AI模型分析文本语义进行匹配

**人工修正**：
- 当匹配结果不确定时，系统会显示AI识别结果和建议
- 点击【确认添加到关键词】可将匹配文本添加为特性关键词
- 点击【人工修正匹配列表】查看所有待修正记录

### 3.7 公式维护

**功能说明**：维护中间数据表中的计算公式，支持从多个数据源引用变量。

![公式维护-主界面](./screenshots/基础管理/03-17-公式维护-主界面.png)

**公式类型**：
- **计算型（CALC）**：通过公式计算得出结果
- **判定型（JUDGE）**：根据条件判定等级
- **只展示（NO）**：仅展示原始值，不计算

**操作步骤**：

1. 进入【基础管理】→【公式维护】

2. **筛选公式**：
   - 使用"来源筛选"过滤系统默认/自定义公式
   - 使用"类型筛选"过滤计算/判定/只展示类型
   - 使用关键字搜索公式名称或列名

3. **新增公式**：
   - 点击【新增公式】按钮
   
   ![公式维护-新增弹窗](./screenshots/基础管理/03-18-公式维护-新增弹窗.png)
   
   - 填写公式名称
   - 选择公式类型
   - 选择数据来源
   - 配置公式表达式或判定规则
   - 设置单位和精度
   - 点击【确定】保存

4. **编辑公式属性**：
   - 点击【编辑】按钮
   - 修改显示名称、单位、精度等属性

5. **编辑公式表达式**：
   - 点击【编辑公式】按钮
   
   ![公式维护-公式构建器](./screenshots/基础管理/03-19-公式维护-公式构建器.png)
   
   - 在公式构建器中配置计算表达式
   - 支持引用其他列、公共维度、常量
   - 点击【保存】应用公式

6. **查看判定规则**：
   - 对于判定型公式，点击【查看判定】
   - 查看各等级的判定条件

7. **删除公式**：
   - 仅自定义公式可删除
   - 系统默认公式无法删除

8. **更新列**：
   - 点击【更新列】按钮同步最新的数据列

**公式语法说明**：
- 列引用：`{columnName}` 或 `{列显示名}`
- 公共维度引用：`@dimensionKey`
- 数学运算：`+`、`-`、`*`、`/`、`^`
- 函数支持：`AVG()`、`MAX()`、`MIN()`、`SUM()`等

### 3.8 判定等级

**功能说明**：为判定型公式配置不同产品规格下的等级判定条件。

![判定等级-主界面](./screenshots/基础管理/03-20-判定等级-主界面.png)

**界面布局**：
- 左侧：产品规格选择 + 判定项目列表
- 右侧：等级配置表格

**操作步骤**：

1. 进入【基础管理】→【判定等级】

2. **选择产品规格**：
   - 在左侧顶部选择产品规格
   - 如不选择则为通用配置

3. **选择判定项目**：
   - 在左侧列表中点击判定项目名称
   - 右侧显示该判定项目的等级配置

4. **新增等级**：
   - 点击【新增等级】按钮
   
   ![判定等级-新增弹窗](./screenshots/基础管理/03-21-判定等级-新增弹窗.png)
   
   - 填写等级名称
   - 设置优先级（数值越小优先级越高）
   - 选择质量状态（合格/不合格/其他）
   - 设置是否参与统计
   - 设置是否为默认等级
   - 选择颜色标识（可选）
   - 点击【确定】保存

5. **配置判定条件**：
   - 点击等级行的【条件】按钮
   
   ![判定等级-条件编辑器](./screenshots/基础管理/03-22-判定等级-条件编辑器.png)
   
   - 在条件编辑器中配置判定规则
   - 支持多条件组合（且/或关系）
   - 条件可基于：列值、外观特性、特性等级等
   - 点击【确定】保存条件

6. **拷贝等级**：
   - 点击【拷贝】按钮复制当前等级
   - 可修改后保存为新等级

7. **批量复制**：
   - 点击【批量复制】按钮
   
   ![判定等级-批量复制弹窗](./screenshots/基础管理/03-23-判定等级-批量复制弹窗.png)
   
   - 选择源产品规格和目标产品规格
   - 一键复制所有等级配置

8. **删除等级**：
   - 点击【删除】按钮确认删除

9. **调整优先级**：
   - 拖动表格行调整等级优先级顺序
   - 优先级高的等级会先被匹配

**条件配置说明**：
- 支持条件组嵌套（组内可含子组）
- 条件类型：列值比较、特性存在、等级匹配等
- 比较运算符：等于、不等于、大于、小于、包含等

   - 模板编码：系统自动生成的唯一标识

   | 列名 | 说明 |
   |------|------|

**使用场景**：

## 六、常见问题

### Q1: 为什么导入的数据显示为"无效数据"？

**可能原因**：
- 炉号格式不正确
- 必填字段缺失
- 数据类型不匹配

**解决方法**：
- 在"原始数据"页签查看具体错误信息
- 修正Excel文件后重新导入

### Q2: 计算公式没有自动计算？

**可能原因**：
- 公式配置错误
- 引用的列值为空
- 公共维度未配置

**解决方法**：
- 检查公式配置是否正确
- 查看计算日志了解具体错误
- 确认引用的公共维度有值

### Q3: 外观特性匹配不准确？

**解决方法**：
- 在"外观特性管理"中完善关键词
- 使用语义匹配测试功能验证
- 在人工修正列表中修正匹配结果

### Q4: 如何修改产品规格的检测列数？

**操作步骤**：
1. 进入【产品规格定义】
2. 找到目标产品规格
3. 点击【编辑】
4. 修改"检测列数"
5. 保存后中间数据表格会自动调整

**注意**：修改检测列数后，已有数据可能需要重新导入

### Q5: 判定结果不符合预期？

**检查项**：
1. 确认判定等级的条件配置正确
2. 检查各等级的优先级顺序
3. 确认外观特性和等级已正确关联
4. 查看中间数据的贴标列是否有值
