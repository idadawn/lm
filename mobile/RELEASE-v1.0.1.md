# 检测室数据分析移动端 v1.0.1 发布说明

**版本：** 1.0.1 (build 101)  
**日期：** 2026-04-27  
**平台：** Android / uni-app

---

## 更新内容

### 1. 个人资料性别显示修复
- 修复后端返回数字性别（1/2/3）时无法正确显示中文的问题
- 新增性别映射：1→男、2→女、3→保密
- 保存时自动将中文转回数字提交给后端

### 2. 修改密码验证码修复
- 修复验证码图片 URL 错误导致不显示的问题
- 接口路径修正为 `/api/oauth/ImageCode/4/...`
- 基础地址改为动态获取（跟随用户切换的服务器地址）

### 3. 接口配置页面统一
- 「我的」页接口配置从弹窗输入改为跳转独立页面
- 与登录页「切换服务器」使用完全相同的 `pages/server/server` 页面
- 支持地址输入、连接测试、保存切换

### 4. 检查更新本地版本修复
- 修复本地版本号显示不正确的问题
- APP 环境下改用 `plus.runtime.getProperty()` 获取准确版本（支持 wgt 热更新后版本）
- 非 APP 环境回退到 `uni.getAppBaseInfo()`

### 5. 首页互动效果增强
- KPI 卡片新增点击缩放反馈 + 轻震动效果（APP）
- 数据卡片新增入场淡入上浮动画
- 质量分布/Top5 进度条新增弹性过渡动画
- 日期选择器、刷新按钮新增按压反馈

### 6. 智能对话页面重构
- 从占位页重写为完整对话页面
- 支持消息气泡列表、底部输入框、快速问题标签
- 接入 SSE 流式对话（`streamNlqChat`）
- 支持展示知识图谱推理链
- 新增加载中打字机动画
