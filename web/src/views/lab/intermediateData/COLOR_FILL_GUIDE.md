# 🎨 表格颜色填充功能使用指南

## 功能简介

中间数据表格现在支持类似WPS的单元格颜色填充功能，使用WPS标准的10个颜色，帮助您更直观地标识和分类数据。

## 🚀 快速开始

### 1. 选择颜色
在表格上方的工具栏中，可以看到一排颜色方块：
- 点击任意颜色即可选中
- 选中的颜色会显示边框高亮

### 2. 填充单元格
选中颜色后：
- 直接点击表格中的任意单元格
- 该单元格会立即填充选中的颜色
- 文字会自动变为白色，确保可读性

### 3. 清除颜色
- 点击工具栏中的"清除"按钮，可以取消当前选中的颜色
- 之后再点击单元格就不会填充颜色了

### 4. 修改颜色
- 选中新的颜色
- 再次点击已填充的单元格，颜色会被新颜色覆盖

## 💡 使用技巧

1. **批量标记**：先选中颜色，然后快速点击多个单元格
2. **分类管理**：不同颜色代表不同的数据状态或类别
3. **快速识别**：通过颜色快速定位关键数据
4. **灵活使用**：可以随时更改或清除颜色

## ⚠️ 注意事项

1. **数据保存**：颜色数据目前只保存在前端内存中，刷新页面后会丢失
2. **颜色覆盖**：重新选择颜色并点击单元格会覆盖原有颜色
3. **所有单元格**：表格中的所有单元格都支持颜色填充功能
4. **样式保留**：填充颜色不会影响单元格的原有内容和样式

## 🔧 技术实现

### 前端状态管理
```typescript
// 当前选中的颜色
const selectedColor = ref<string>('')

// 存储所有已填充颜色的单元格
const coloredCells = ref<Record<string, string>>({})
```

### 核心函数
```typescript
// 获取单元格颜色
function getCellColor(rowId: string, field: string): string {
  return coloredCells.value[`${rowId}-${field}`] || ''
}

// 处理单元格颜色填充
function handleCellColor(rowId: string, field: string) {
  if (selectedColor.value) {
    const key = `${rowId}-${field}`
    coloredCells.value[key] = selectedColor.value
  }
}
```

## 📋 实际应用场景

1. **质量检测**：用红色标记不合格数据，绿色标记合格数据

3. **数据分类**：用颜色区分不同的产品批次或生产线
4. **异常标记**：快速标记需要关注的数据
