# 表格颜色填充功能说明

## 功能概述

中间数据表格现在支持类似WPS的单元格颜色填充功能，使用WPS标准的10个颜色。

## 使用方法

### 1. 选择颜色
在表格上方的工具栏中，可以看到一排颜色方块，点击其中一个颜色即可选中。

### 2. 填充单元格
选中颜色后，直接点击表格中的任意单元格，即可为该单元格填充选中的颜色。

### 3. 清除颜色
点击工具栏中的"清除"按钮，可以取消当前选中的颜色。之后再点击单元格就不会填充颜色了。

### 4. 重新填充
如果需要更改单元格颜色，只需选中新的颜色，然后再次点击该单元格即可。

## 颜色列表

10个标准WPS颜色：
- #C00000 - 深红
- #FF0000 - 红色
- #FFC000 - 橙色
- #FFFF00 - 黄色
- #92D050 - 浅绿
- #00B050 - 绿色
- #00B0F0 - 天蓝
- #0070C0 - 蓝色
- #002060 - 深蓝
- #7030A0 - 紫色

## 注意事项

1. 颜色数据目前只保存在前端内存中，刷新页面后会丢失
2. 所有单元格都支持颜色填充，包括可编辑单元格
3. 填充颜色不会影响单元格的原有内容和样式（如负数红色显示仍然有效）
